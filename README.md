# springboot_vue_test

## Project setup
### cd 进入项目文件夹，使用如下命令
```
npm install
```

### 安装完后使用如下命令开启前端网页
```
npm run serve
```

### 打包
```
npm run build
```

##后端调试：
导入maven项目，刷新、clean、install，等待安装所有依赖即可

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).
